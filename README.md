# LingXi's Giscus Comment Appearance Themes

✨ This is LingXi9374's Giscus comment system appearance theme repository. You can use it with the help of **jsDelivr CDN**.

-----

👉 Usage:

📋 Copy the CSS theme name (such as `"Acrylic_frosted_glass_theme"`) from the repository, paste it, and replace the URL variable example given below.

```javascript
<script src="https://giscus.app/client.js"
  ...
  data-theme: "https://fastly.jsdelivr.net/gh/LingXi9374/giscus-themes/<themes>.css"
  ...
  async>
</script>

// Replace into

<script src="https://giscus.app/client.js"
  ...
  data-theme: "https://fastly.jsdelivr.net/gh/LingXi9374/giscus-themes/Acrylic_frosted_glass_theme.css"
  ...
  async>
</script>
```

🤔 Different blog front-end engines may have different configurations; please refer to the official documentation for modifications.

-----

💎 If you see a change in the appearance of the Gicus comments section, then you've succeeded!

☕ Enjoy!
